# NexHackPortal

A comprehensive Hackathon Management System featuring team formation, QR-based food coupon redemption, and role-based dashboards for participants and organizers.

## Made by Manish

## Features

- **Team Formation** - Find teammates easily through the portal
- **QR-Based Food Coupons** - Streamlined meal redemption system
- **Role-Based Dashboards** - Separate views for participants and organizers
- **Event Schedule** - Track hackathon events and workshops
- **User Authentication** - Secure login and registration system

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Recharts
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```
